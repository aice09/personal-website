# Personal Website

My personal site and blog — built with Jekyll, deployed on Vercel.

🔗 **Live site:** [personal-website-tau-gules-15.vercel.app](https://personal-website-tau-gules-15.vercel.app/)

## About

This is where I write about infrastructure work, homelab experiments, and things I'm learning along the way — mostly Kubernetes, OpenStack, and networking.

## Tech stack

- **[Jekyll](https://jekyllrb.com/)** — static site generator
- **[Minima](https://github.com/jekyll/minima)** theme
- **[Vercel](https://vercel.com/)** — hosting and CI/CD, auto-deploys on push to `main`
